Data Visualization Dashboard

Using the perks of Visualization this project aims to analyze opinions of individuals from surveys and tweets to provide perception on the variations in trends .

Group 5

Historical Survey Dataset

This Dashboard contains the trend analysis of the survey that was collected from some previous college experiment to know the opinions about online vs Offline learning.

Live Survey Dataset

We created a Google form to collect opinions from current students and based on that thrend analysis was perform to identify the opinions on online learning vs offline learning.

Scrapped Twitter Dataset

We scrapped recent tweets and performed sentiment analysis and emotion analysis to visualize trends among twitter users with respect to online and offline learning.

Historical Twitter Dataset

We found a set of tweets of 2020 regarding online education which also has location details using that we visualize the trends among them in 2020.
